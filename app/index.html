What is Krane? Try It!
 

Multicloud

Create and destroy Docker-based installations (or ships as we like to call them) seamlessly across multiple cloud platforms, including AWS, Digital Ocean and Flexiant.

100% Docker

Code is based on Docker's code, and so is 100% Docker compatible and 100% open source. Think of it as an extension to Docker.

Management

Securely manage and control containers as well as ships (the VMs they run in) across multiple clouds and locations, bringing Docker to a whole new level.
